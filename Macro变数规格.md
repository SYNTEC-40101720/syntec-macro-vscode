MACRO可使用的变数主要可分为区域变数(Local variable, #1~#400)、系统变数(System variable, #1000~#31986)，以及公用变数(Global variable, @1~@165535)三种。

不同型态的变数将有其不同的生命周期及读写规则，以下各章节将详细说明。

## 公用变数(Global variable)

| **编号** | **说明** | **规则** | **型态** |
| --- | --- | --- | --- |
| **@0** | 空白VACANT | R   | \-  |
| **@1~ @400** | 一般运算用变数 | R/W | Double |
| **@401~@655** | 对应寄存器 R1~R255 | R/W | Long |
| **@656~@999** | 有记忆性变数(关机后仍存在) | R/W | Double |
| **@1000** | 系统变数，请勿使用 (版本: 10.114.x/10.116.x)<br><br>- 软体版本 10.118.56Z, 10.118.60T, 10.118.66M, 10.118.70G, 10.118.77 以后，已改为有记忆性变数(关机后仍存在)。 | R   | Double |
| **@1001~@1999** | 有记忆性变数(关机后仍存在) | R/W | Double |
| **@10000~@14095** | 对应寄存器 R0~R4095 | R/W | Long |
| **@60000~@79999** | 扩充公用变数(请配合No3813使用，此外仅适用于CE系统) | R/W | Double |
| **@100000~@165535** | 对应寄存器 R0~R65535 | R/W | Long |
| **备注** | - 所有公用变数除非遇到关机，否则并无生命周期之限制。<br>- @1~@400关机后若仍需要记忆的话，请利用No3811设定之。<br>- DOS仅支援R0~R7999，因此使用对应公用变数时需特别注意。<br>- @值对应寄存器的部分，可写入之范围如下：<br>  - R50~R80、R101~R511、R1024~R4095、R5800~R7999、R10000~R10999、R15000~R65535 |     |     |
