1 G10L10000-圖形模擬範圍設定
1.1 指令格式
G10 L10000 X_ I_ Y_ J_ Z_ K_
X、I：X軸範圍邊界
Y、J：Y軸範圍邊界
Z、K：Z軸範圍邊界
1.2 說明
設定圖形模擬的範圍
1. 此命令被讀取後，會立即設定圖形模擬的邊界，並開始繪製路徑。
2. 使用此命令可節省系統判斷圖形模擬邊界的時間。
1.3 注意事項
1. 此命令越早被讀取，節省的時間越多，因此建議把此命令放在加工檔的第一行。
2. 此命令不需要Cycle Start即可生效。因為系統讀取加工檔進行模擬的動作，在打開圖形模擬時即
會開始。
3. 如果引數帶有小數點，則該引數單位為 IU；如果輸入引數未帶有小數點，則該引數單位為 LIU。
4. 此功能引數的單位受 Pr3241 影響。
5. 範圍邊界之引數必須成對，否則系統將發出警報 COR-023 語義錯誤。
EX：G10 L10000 X-0.1 I0.1 Y-0.2 J0.2 Z1 // Z軸邊界引數沒有成對，系統發出警報。
6. 至少需指定兩個軸的範圍邊界，否則系統將發出警報 COR-023 語義錯誤。
EX：G10 L10000 X-0.1 I0.1 // 只有指定X軸邊界，系統發出警報。
7. 不支援使用 G10 L1000 修改暫存器數值後，再使用該暫存器來設定範圍邊界。
EX：
若一開始R500 = 50
G10 L1000 P500 R1000; // 使用 G10 L1000 修改 R500 = 1000
G10 L10000 X-0.1 I@100500 Y0. J3.; // X軸範圍邊界值仍是 50
1.4 程式範例
G10 L10000 X0. I1. Y-1. J1. Z0. K1.; // 將圖形模擬範圍定於 X = 0~1, Y = -1~1, Z = 0~1 的空間中

2 G10L1000-R寄存器写入
2.1 指令格式
G10 L1000 P_ R_;
P：R bit號碼
R：寫入R bit的數值
2.2 說明
1. 透過G碼變更R bit的數值
2. 在绝对指令模式（G90）下是绝对值，在增量指令模式下（G91）下是增量值。
3. 在G62模式下指令会准确在G01/G02/G03单节交接处发出指令。

42 G10L900-启动轴耦合功能
42.1 指令格式
G10 L900 P_ Q_ R_ I_ J_ K_;
P：指定主动轴，可指定轴号或轴名称。
Q：指定从动轴，可指定轴号或轴名称。
R：指定从动轴分量，同时代表启动耦合或解除耦合，有下R引数表示启动耦合，没有下R引数表示解除
耦合。
I：指定主动轴分量（可不下，初始值为1）
J：指定耦合时间，单位：ms（可不下，初始值为0）
K：指定耦合分离时间，单位：ms（可不下，初始值为0）
42.2 说明
使用G码启动轴耦合功能
42.3 注意事项
1. 启动之轴耦合型态必为主从耦合。
2. 耦合轴之轴名称或轴号需为已开启之轴向，否则会发COR-105警报。
3. 若指定没开启之轴向进行耦合，不会发出警报，且不会影响到其他组的耦合关系。
4. 已开启之主动轴不可再指定为其它组耦合的从动轴。
5. G码指令执行完即耦合；解除后该组耦合即消除。
6. 跨轴群耦合时，请使用 G04.1 来完成轴群间同步，不同轴群同时对同一轴下命令，则未作座标同
步情况下将有机会导致座标异常。
7. 相同主动轴与从动轴组合只能存在一组。
8. Reset后会解除所有以G码启动之轴耦合。
9. 轴耦合最大组数为16组，包含参数设定之轴耦合，若组数超过会发COR-97警报。
10. 主动轴分量不可设为0，否则发出警报COR-105。
11. 软件版本10.116.x之后始支援 I 引数。
12. 绝对座标、机械座标、相对座标、剩余距离的显示规格如下：
a. 若主动轴与从动轴在同一轴群，则从动轴的绝对座标、机械座标、相对座标、剩余距离皆会
显示。
b. 若主动轴与从动轴「未」在同一轴群，则从动轴的绝对座标、机械座标、相对座标会显示，
但剩余距离「不显示（为0）」。

43 G10L901-修改轴耦合参数设定
43.1 指令格式
G10 L901 P_ Q_ R_ I_ J_ K_;
L：指定欲修改耦合设定之组别，可设定为901~904
P：设定主动轴轴号
Q：设定从动轴轴号
R：设定从动轴分量
I：设定主动轴分量
J：设定耦合完成时间，单位：ms
K：设定耦合分离时间，单位：ms
43.2 说明
使用G码修改轴耦合参数设定
43.3 注意事项
1. 若该组轴耦合参数未设定，此功能无效。
2. 不需要修改之耦合设定，其对应引数可以不指定。
3. 目前不支援修改轴耦合型态。
4. 若原型态为一对多，从动轴的选取以Bit决定，Bit on 表示该轴加入耦合，设定方式如下：
Bit 1：X轴，换算成十进位为2；
Bit 2：Y轴，换算成十进位为4；
Bit 3：Z轴，换算成十进位为8；
Bit 4：第四轴，换算成十进位为16，其余轴向依此类推。
当Pr3822设定为28（28=4+8+16）时，表示设定Y轴、Z轴与第四轴为从动轴。
5. 使用此功能修改耦合设定，不会取代参数，而是直接套用L引数后方引数内容。
6. Reset后，此功能之设定不会保留，而是回复原本的参数设定。
7. 主动轴分量不可设为0，否则发出警报COR-105。
8. 使用此功能前，务必先解除耦合状态，否则耦合设定直接被修改，会导致命令突然改变，可能发
生预料外的动作。
9. 目前不支援修改机构耦合参数设定
43.4 程序范例
G10 L901 Q4 R2;
// 修改第1组轴耦合设定，主动轴不变，从动轴改为第4轴
// 从动轴分量改为2
